# ❤️ CoupleHub

CoupleHub is a private app designed for couples to stay emotionally connected every day.

It combines mood sharing, love messages, shared events, and wishlist planning into one simple and intimate space.

## ✨ Features

### 😊 Mood Check-In
- Each partner selects their daily mood using emoji
- Real-time sync between partners
- Mood history stored per day
- Visible inside the app and on home screen widget

### 💕 Quick Love Messages
- Send instant “Miss you” and “Thinking of you” messages
- Real-time delivery
- Inbox with unread badge
- Message history

### 📅 Shared Calendar
- Add one-time or yearly recurring events
- Event reminders for today and tomorrow
- Shared couple timeline

### 🎁 Wishlist
- Personal and partner wishlists
- Categories (Movies, Travel, Gifts, Other)
- Mark items as completed
- Edit & delete anytime

### 🧩 Home Screen Widget
- Displays both partners’ current moods
- Shows today’s event reminder
- Quick “Thinking of you” button
- Designed to match app theme

## 🔒 Privacy First
- Private couple space
- Secure Firebase Authentication
- Firestore security rules ensure only partners can access data

## 🛠 Built With
- Kotlin
- Jetpack Compose
- Firebase Auth
- Cloud Firestore
- Android App Widgets

Active personal project — continuously improving features and UI.
